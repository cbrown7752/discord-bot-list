Install the python package discord.py

Run pip install from your system terminal/shell/command prompt.

m pip install discord.py

Run the sample code

Replace the token value with the token from your bot that you saved earlier.

# Work with Python 3.6 or above
import discord


bot = discord.Client()


@bot.event
async def on_ready():
    print('Logged in as')
    print(client.user.name)
    print(client.user.id)
    print('------')

bot.run('PUT_YOUR_TOKEN_HEAR')
